Technologies Used

An overview of the technologies used in our COVID-19 Resources project.

D3.js

D3.js (Data-Driven Documents) is a JavaScript library for producing dynamic, interactive data visualizations in web browsers. It uses HTML, SVG, and CSS to bring data to life.

Key Features:

  • Data binding
  • Scalable Vector Graphics (SVG) support
  • DOM manipulation
  • Animations and transitions

Learn more about D3.js at the official D3.js website.

Bootstrap

Bootstrap is a popular open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Key Features:

  • Responsive grid system
  • Pre-styled components
  • JavaScript plugins
  • Customization

Learn more about Bootstrap at the official Bootstrap website.

Axios

Axios is a promise-based HTTP client for JavaScript. It is used to make requests to APIs and handle responses, making it easy to fetch data asynchronously.

Key Features:

  • Promise-based for cleaner code
  • Supports the async/await syntax
  • Automatic JSON data transformation
  • Interceptors for request and response

Learn more about Axios at the official Axios website.

jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, and animation much simpler with an easy-to-use API that works across a multitude of browsers.

Key Features:

  • DOM manipulation
  • Event handling
  • Ajax calls
  • Utilities for working with arrays and objects

Learn more about jQuery at the official jQuery website.
